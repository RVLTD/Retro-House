# Retro House

O **Retro House** é um site de notícias, guias e análises focado na cena de Retro Gaming e consoles portáteis modernos. O projeto foi desenvolvido como um estudo aprofundado de Front-end, focando em layouts modernos, semântica HTML e responsividade.

## Tecnologias, Ferramentas e Conceitos Utilizados

Neste projeto, apliquei conceitos fundamentais e avançados de desenvolvimento Web:
* Editor: [Visual Studio Code](https://code.visualstudio.com/)
* **HTML5 Semântico:** Uso rigoroso de tags como `<header>`, `<main>`, `<section>`, `<article>` e `<footer>` para melhor acessibilidade e SEO.
* **CSS3 Moderno:** * **CSS Grid:** Implementação de um layout **Bento Grid** para a seção de análises.
    * **Flexbox:** Utilizado para alinhamento de itens no cabeçalho, navegação e listas de guias.
    * **Responsividade:** Uso de Media Queries para adaptação de layouts em dispositivos móveis.
    * **Unidades Relativas:** Uso de `rem` para tipografia, garantindo acessibilidade e escalabilidade.
* **Interatividade:** Efeitos de hover com `transitions` e `transforms` para melhorar a experiência do usuário (UX).
* **Fontes Personalizadas:** Integração de fontes externas via `@font-face` para reforçar a identidade visual "Retro".
